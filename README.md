## 购买机场前需要知道

### 1. 买机场和买基金一样

当前表现不代表未来表现。因为机场很快线路和稳定性。然而线路总是会有各种问题。

### 2. 线路比流量更重要

一般来说专线机场体验最好。专线主要是不过墙延迟低，网络波动小，又专门出口，专线机场日常使用体验会更丝滑更舒服。机场比的是钞能力，速度要看机场的总带宽大小，稳定要看机场的上游的线路质量，解锁看的是机场的落地/解锁商是否优秀。机场带宽大和稳定性最重要。开得久的大机场也有处理各种情况的经验，比如经常出现的域名被墙。202604：由于近期国内中转被大规模打击清退，且目前还在持续打击中，因此很多机场无国内中转可用，大部分机场在使用路由好的海外机器做中转，也是直连或者是使用透传专线（避免被通报），那如果是电信网，则推荐购买有三网优化或者电信优化的直连的机场，否则体验会很差。         

### 3. 不要一次买很多年

一下别买太长时间，谁也不知道机场什么时候跑路。TG过于高调的机场也需要谨慎购买

### 4. 一定准备备用机场

备用很重要，如果经常需要科学上网，那么就不能只有一个机场，特别是202604之后大规模清理国内中转机之后很多机场都不稳定，因此高强度翻墙的推荐买多个机场：有多个中转入口的机场或者买两家不同入口的机场，避免失联。

### 5. 不推荐超便宜机场

不推荐购买太便宜的机场，很多套路，注意避开：1)一开始很便宜或搞超大折扣活动，AFF比例非常高。低价吸引一波来割韭菜，捞一波就跑。别看单价流量多就觉得值，不好用也是浪费钱。 2)使用便宜月抛直连线路，成本低。线路变化频繁，随时一键跑路。 3)线路多但复用，多条线路但入口和落地其实是同一条。 很多便宜机场优点只有便宜，稳定性和速度一言难尽。 买多了几家机场就知道：机场便宜不好用，好用不便宜。

## FAQ

### 哪种机场最稳定？

一般价格高，线路多的机场比较稳定，因为机场老板更舍得花钱买线路。而且价格高了用户要求高也会倒逼机场老板优化线路。价格便宜的机场预算有限总是会在购买线路上节省。

### 为什么推荐多个机场？

因为机场本质是服务器，肯定会挂的。所以推荐购买2个机场以上。

### 为什么很多机场开始用AnyTLS？

1）202606之前Anytls不会被通报。 2）202604开始大规模通报国内入口中转，使用Anytls协议可以使用直连。是目前比较适合的协议之一。

### 延迟低是不是速度快？

在带宽和占用正常的情况下：延迟低和速度无关。延迟主要是路程有关，如果是在南方，一般香港节点延迟是最低的。但是如果入口占用高，那么延迟就会高。这种一般速度也会低。

### 为什么机场官网打不开？

因为防火墙会经常污染机场官网。

### 为什么测速快实际体验不好？

测速可能是白天测速的，晚高峰使用的人多带宽不足。

测速可能使用的是移动测速，但是你是电信网络，线路跨网限速不好用。

### 为什么IEPL更贵？

IEPL专线由于是内网传输，不走公网，不经过防火墙，采购价格就非常高，所以机场会延迟低价格贵。

### 为什么公网中转延迟高？

公网传输，而不是内网传输。会绕路，导致延迟较高。机场主要是看速度和稳定性。延迟高一点其实影响不大。

### 为什么封订阅推荐使用官方客户端？

目前由于通报严重，很多机场为了避免节点泄露，保护线路保护节点。所以进行封端。
